# Claude Code Startup

## Instructions

Before writing any code:

1. Read `CLAUDE.md`.
2. Read every document under `docs/`.
3. Read:
   - `management/PROJECT_ROADMAP.md`
   - `management/TASKS.md`
   - `management/CHANGELOG.md`
4. Identify the active Epic under `epics/`.
5. Determine:
   - Current milestone
   - Current Epic
   - Highest-priority unfinished task
   - Dependencies
   - Files likely to be modified
6. Present a concise implementation plan.
7. Wait for approval before modifying code.

## After completing a task

- Update `management/TASKS.md`.
- Update `management/CHANGELOG.md`.
- Summarize modified files.
- Recommend the next logical task.
- Stop after completing one coherent feature unless instructed otherwise.
