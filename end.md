# Claude Code – End of Development Session

Before ending the current development session, complete the following tasks.

## 1. Review Today's Work

Summarize:

- Features completed
- Bugs fixed
- Refactoring performed
- Documentation updated
- Tests executed
- Outstanding issues

---

## 2. Update Project Management

Update:

### management/TASKS.md

- Mark completed tasks.
- Add newly discovered tasks.
- Reprioritize remaining tasks.
- Move completed tasks into the Completed section.

### management/CHANGELOG.md

Record:

- Features added
- Improvements
- Bug fixes
- Refactoring
- Documentation changes

Use today's date.

---

## 3. Documentation Review

If implementation changed any of the following, update them:

- docs/SRS.md
- docs/02-Architecture.md
- docs/03-Database.md
- docs/04-API.md
- docs/05-UIUX.md
- docs/06-CodingStandards.md
- docs/07-TestingStrategy.md
- docs/08-DeploymentGuide.md
- docs/09-CatalogDataModel.md
- docs/10-DeveloperGuide.md

Update only if necessary.

---

## 4. Code Quality Review

Verify:

- No unnecessary code duplication
- No unused imports
- No dead code
- No TODOs without explanation
- Consistent naming
- Consistent formatting
- Lint passes
- Build succeeds

---

## 5. Testing

If applicable, verify:

- Unit tests
- Integration tests
- API tests
- Component tests
- End-to-end tests

Summarize any failures.

---

## 6. Git Summary

Prepare a summary suitable for a commit.

Include:

- Files modified
- Files created
- Files deleted

Suggest a Conventional Commit message.

Example:

feat: implement curriculum roadmap visualization

---

## 7. Project Status

Report:

Current Milestone

Current Epic

Current Completion Estimate

Example:

Milestone:
Frontend Development

Epic:
006 – Curriculum Roadmap

Overall Progress:
42%

---

## 8. Remaining Work

List:

- Highest-priority unfinished tasks
- Blockers
- Risks
- Dependencies

---

## 9. Recommend Next Session

Suggest the single highest-value task for the next development session.

Do not begin implementation.

Wait for user approval.

---

## 10. Final Summary

Provide a concise report including:

- What was accomplished
- What documentation changed
- What remains
- Recommended next step

End the session after presenting the summary.
